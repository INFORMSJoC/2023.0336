# Files

- `stack_nolhs.rb` generates a design of experiments from an input set of parameters
- `stack_nolhs_wrapper.py` a wrapper for `stack_nolhs.rb` that adds support for headers
- `run_yaml_design.rb` reformats an input csv design of experiments file according to an input yaml file format

See `sensitivity/parameters.csv` for an example input, `sensitivity/stack_nolhs_wrapper.yml` for an example run configuration, and `sensitivity/README.md` for an example set of instructions.
